Cat Monitoring System

Time to apply what I have learned!

The goal of the project is to build a self-sustaining system to monitor your cats 24/7 to see what shinanigens they commit. It is a 24/7 real time monitoring system for cats. The system is powered by renewable energy, and connects via WiFi or Bluetooth. It will notify the user of relevant activities and save recordings. I'll try to implement this in five phases.

  • 1. Basic Detection: First lets actually implement an object detection system on the computer.
  • 2. App and website: Make an app and website that can implement the object detection system.
  • 3. LLM: Implement an llm that will create a summary of important events detected and notify users.
  • 4. Database: Clips of the cat will be created and saved in a database.
  • 5. Hardware: Migrate everything to new hardware, physical cameras and use an esp32 or a raspberry as control.
